# Mirai – Smart Elderly Health Assistant

Mirai is a Python-based health compliance and reminder system designed to help elderly individuals maintain consistent medication and supplement routines.

This project was developed as part of the Ontario-wide Mirai Innovation Challenge and earned **3rd Place** for its technical design and social impact.

## Features
- Medication & supplement reminders
- Compliance tracking (in development)
- Risk detection (planned)
- Secure data storage (planned)

## Tech Stack
- Python
- SQLite (planned)
- CLI Interface
