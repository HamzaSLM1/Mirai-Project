# Mirai â€“ Smart Elderly Health Assistant

Mirai is a smart health monitoring and reminder system designed to support elderly individuals in maintaining consistent medication and supplement routines.

This project was created as part of the Ontario-wide Mirai Innovation Challenge, where it earned **3rd Place** for its technical design and social impact.

Mirai combines:
- A Python-powered reminder engine
- Health tracking logic
- Risk detection rules
- A business and implementation model

The system is designed to demonstrate how software, automation, and data logic can be used to improve real-world healthcare outcomes.

## Planned Modules

| Module | Purpose |
|------|--------|
| reminder_engine.py | Schedules and sends reminders |
| tracker.py | Tracks supplement and medication compliance |
| risk_engine.py | Detects missed-dose patterns |
| dashboard.py | Displays health compliance summaries |
